Pin Definitions:
5V   -   5V Power
3V3  - 3.3V Power
GND  - Ground Connection
LED1 - On Board LED1 - Power Indicator01
LED2 - On Board LED2 - Set to OSC/4000000 - blink led to show CPLD is working
LED3 - On Board LED3 - Set to OSC/2000000 - blink led to show CPLD is working
SSC  - Synchronous Serial Clock - DTR
SSD  - Synchronous Serial Data  - DSR
XOSC - External Crystal Oscillator Monitor (50MHz)
OSC  - Oscillator Used by Shift Registers

Smoothie Pins:
SG   - Smoothie Ground
S5V  - Smoothie 5V from one of the limit switches
SA+  - Smoothieboard Input  A+ Limit
SA-  - Smoothieboard Input  A- Limit
SB+  - Smoothieboard Input  B+ Limit
SB-  - Smoothieboard Input  B- Limit
SC+  - Smoothieboard Input  C+ Limit
SC-  - Smoothieboard Input  C- Limit
SAS  - Smoothieboard Output A Step
SAD  - Smoothieboard Output A Direction
SBS  - Smoothieboard Output B Step
SBD  - Smoothieboard Output B Direction
SCS  - Smoothieboard Output C Step
SCD  - Smoothieboard Output C Direction
SInk - Smoothieboard Ink Solenoid Signal
SLsr - Smoothieboard Laser PWM Control Signal
SKil - Smoothieboard Kill Signal
SPau - Smoothieboard Pause Signal


Machine Control Pins:
A-L  - Machine A -Limit Switch - Connect to ground during limit
A+L  - Machine A +Limit Switch - Connect to ground during limit
B-L  - Machine B -Limit Switch - Connect to ground during limit
B+L  - Machine B +Limit Switch - Connect to ground during limit
C-L  - Machine C -Limit Switch - Connect to ground during limit
C+L  - Machine C +Limit Switch - Connect to ground during limit
AS+  - Driver A Step+ - for rising  edge drivers or the + side of differential step signal
AS-  - Driver A Step- - for falling edge drivers or the - side of differential step signal
BS+  - Driver B Step+ - for rising  edge drivers or the + side of differential step signal
BS-  - Driver B Step- - for falling edge drivers or the - side of differential step signal
CS+  - Driver C Step+ - for rising  edge drivers or tymhe + side of differential step signal
CS-  - Driver C Step- - for falling edge drivers or the - side of differential step signal
AS+  - Driver A Direction+ - for rising  edge drivers or the + side of differential Direction signal
AS-  - Driver A Direction- - for falling edge drivers or the - side of differential Direction signal
BS+  - Driver B Direction+ - for rising  edge drivers or the + side of differential Direction signal
BS-  - Driver B Direction- - for falling edge drivers or the - side of differential Direction signal
CS+  - Driver C Direction+ - for rising  edge drivers or the + side of differential Direction signal
CS-  - Driver C Direction- - for falling edge drivers or the - side of differential Direction signal
Kill - Kill Switch Input
Paus - Pause Switch Input
Lasr - Laser PWM Power Output
Ink  - Ink Solenoid Output
